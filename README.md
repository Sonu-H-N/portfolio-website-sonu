# 🌐 Sonu H N — Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase skills, projects, and contact information.

---

## ✨ Features

- 👨‍💻 Personal introduction (Hero section)
- 🧠 Skills section with tech stack
- 📁 Projects showcase with GitHub links
- 📱 Responsive design for all devices
- 🌙 Dark/Light theme toggle
- 📄 Resume download button
- 📬 Contact section

---

## 🗂 Project Structure

portfolio/
├── index.html # Main webpage
├── style.css # Styling and layout
├── script.js # Interactivity (theme toggle, etc.)
├── profile.jpg # Profile image
└── resume.pdf # Downloadable resume


---

## 🚀 How to Run

### 1. Open locally
Just open `index.html` in your browser.

### 2. Using VS Code
- Install **Live Server**
- Right-click `index.html` → *Open with Live Server*
