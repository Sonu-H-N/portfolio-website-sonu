# Sonu H N — Portfolio Website v2.0

A modern, fully upgraded personal portfolio website.

## 🚀 What's New in v2.0

### ✨ Features Added
- **Preloader** — Animated loading screen before content appears
- **Custom Cursor** — Sleek dot + ring cursor with hover expansion effect
- **Animated Typing Effect** — Loops through multiple roles (Developer, Creator, etc.)
- **Dark / Light Mode** — Persistent toggle saved to localStorage
- **Sticky Navbar** — Frosted glass effect on scroll + active link highlighting
- **Hamburger Menu** — Smooth mobile navigation with animated icon
- **Scroll Reveal Animations** — Sections fade in as you scroll (IntersectionObserver)
- **Skill Bar Animations** — Progress bars animate when scrolled into view
- **Skills Tabs** — Switch between Technical Skills and Tools
- **Improved Projects Section** — Cards with top-border reveal, badges, tags, GitHub/Live links
- **Working Contact Form** — Full client-side validation (name, email, message)
- **Social Links** — GitHub, Email, LinkedIn in Hero and Footer
- **Scroll-to-Top Button** — Appears after scrolling, smooth return
- **Open to Work Badge** — Status indicator on profile image
- **SEO Meta Tags** — Description, keywords, author, Open Graph
- **Favicon** — Emoji favicon for browser tab
- **Responsive Design** — Full mobile/tablet/desktop support
- **Custom Scrollbar** — Styled to match the accent color

### 🎨 Design Improvements
- New typography: **Syne** (display) + **DM Sans** (body)
- CSS Custom Properties (variables) for full theme control
- Cohesive dark-first color system with light mode toggle
- Grid-based hero layout with profile image ring animation
- About section with facts, service cards
- Smooth hover transitions throughout

### 🛠 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Font Awesome 6 (icons)
- Google Fonts (Syne, DM Sans)
- IntersectionObserver API
- CSS Grid & Flexbox
- localStorage for theme persistence
